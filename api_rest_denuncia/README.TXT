Api Rest PHP MySql POO, MVC, PDO

Api sencilla para hacer CRUD basico

GET: Es utilizado únicamente para consultar información al servidor
POST: Es utilizado para solicitar la creación de un nuevo registro. 
PUT: Se utiliza para actualizar por completo un registro existente. 
DELETE: Este método se utiliza para eliminar un registro.

Rutas
GET/    - https://localhost/api_rest_php/api.php            - obtengo todos los registros
GET/1   - https://localhost/api_rest_php/api.php?value=1    - obtengo 1 registro
POST/   -  https://localhost/api_rest_php/api.php           - guardo un registro 
PUT/    - https://localhost/api_rest_php/api.php            - actualiza un registro
DELETE/1 - https://localhost/api_rest_php/api.php?value=12 - elimino 1 registro
